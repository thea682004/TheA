#Giới thiệu về two-way data binding

Dùng để tự động đồng bộ giá trị biến 2 chiều giữa code JS và HTML

Nghĩa là khi html thay đổi giá trị thì code JS tự nhận giá trị mới và khi code JS thay đổi giá trị thì HTMl tự dộng hiển thị giá trị mới
